Pengenalan HTML

HTML adalah dasar pemograman dalam membangun website. Html merupakan singkan dari hypertext markup language. Ada berbagai macam tag element yang dapat digunakan seperti heading dan pragraf

Dalam pembuatan halaman web, HTML dapat dikolaborasikan dengan bahasa lain seperti javascribe dan css (cascading style sheet). css adalah bahasa pemrograman khusus digunakan untuk memberindah tampilan web
